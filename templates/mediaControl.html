Commands

  • next: next track
  • previous: previous track
  • play: start playback
  • pause: pause playback
  • mute: mute volume
  • volume 25: set volume to 25%
  • volume 50: set volume to 50%
  • volume 75: set volume to 75%
  • max volume: set volume to 100%
  • save: add to selected playlist
  • exit: disable voice commands
